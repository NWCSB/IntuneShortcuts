1. Make changes in apps.csv, upload logos in Icons folder
2. Increment the version number on Install, Uninstall, and Detection.ps1 by 1
3. Download the whole repo as zip
4. Unzip and run Create-ScriptPackage.ps1
5. Enter 'install' as the script you want to package
6. Upload to Intune as a new app with the new version number, set it to supercede the previous version
